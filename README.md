## About

A conversion-focused landing page for an Israeli event-photography service that prints custom magnets at weddings, corporate events, birthdays, parties and special occasions. Built mobile-first with full RTL support, an in-page language toggle, a Three.js logo over a looping hero video, and a WhatsApp-first contact flow.

### Sections

| Section | What it shows |
| --- | --- |
| **Hero** | 3D AE logo over a hero video freeze-frame, kicker, headline, body copy. |
| **Gallery** | Sample magnets from real events. |
| **Event types** | Weddings, corporate, birthdays, parties, special occasions. |
| **AI features** | Automatic photo enhancement — lighting, eye-opening, skin tone. |
| **FAQ** | Equipment, printer fallback, AI workflow, design, lead time, coverage area, gallery delivery. |
| **Contact** | Direct WhatsApp button + a form that pre-fills a WhatsApp message. |

## Languages

Hebrew (RTL), Russian, English — full UI and copy translation. Language choice persists across reloads.

## Accessibility

Built-in accessibility widget: text-size steps, high contrast, readable font, underline links, highlight headings, stop motion, big cursor. Settings persist on the device.

## Tech Stack

React 18 · Vite 5 · TypeScript · Tailwind CSS 3 · Framer Motion 11 · three.js · @react-three/fiber · @react-three/drei
